## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/Timefaze
```

Go to the project directory

```bash
  cd Timefaze
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
